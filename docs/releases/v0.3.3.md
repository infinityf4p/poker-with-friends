# Poker Infinity v0.3.3

## Highlights

- The PWA no longer forces `portrait-primary`; installed and browser-launched sessions now follow the device orientation.
- Desktop and landscape table layouts remain unchanged, while players can still rotate to the compact portrait layout when they prefer it.
- A manifest regression test prevents a future release from accidentally locking the site to portrait mode again.

## Verification

- The web test suite validates the manifest orientation in addition to the existing responsive layout and PWA cache boundaries.
- Production browser verification covers the 1470px desktop lobby and table layouts plus the manifest served by the live domain.

## Database and rollback

This release does not add a database migration. Rolling back to v0.3.2 does not require a database rollback, but installed PWA sessions may be locked to portrait mode again.
