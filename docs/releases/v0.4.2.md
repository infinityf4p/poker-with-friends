# Poker with Friends v0.4.2

## 变更

- 牌桌界面只保留状态、数据和操作，删除常驻规则解释、操作提示及重复说明。
- 移除公共牌桌投影和运行时状态中的自由文案字段，避免阵容变化等历史提示持续显示。
- 加载旧版加密快照时自动丢弃遗留文案；数据库迁移同步清理公共快照中的旧字段。
- 邀请页改用人数、盲注和起始筹码三项结构化数据。
- 输入校验和操作失败仍在触发后显示完整错误信息。

## 验证

- 格式、类型检查、单元测试、属性测试与生产构建通过。
- 真实 PostgreSQL、HTTP 和 Socket.IO 多人牌局流程通过。
- 桌面端和移动端完成登录、大厅、邀请及牌桌状态验收。

## 迁移与回滚

- `0003_remove_room_message.sql` 只删除 `rooms.public_snapshot` 中已停用的 `message` 键，不修改表结构或牌局数据。
- 回滚应用时可切换到上一不可变镜像；已清理的旧提示文案不会恢复，也不影响旧版读取其他牌桌字段。
