# Poker Infinity v0.3.2

## Highlights

- The home screen is now a shared table lobby. Every signed-in account can see every open table, current mode, blinds, hand progress, player count, online count, available seats, and current participants.
- Players can join any table directly from its lobby card. Existing invitation links remain compatible, but they are no longer required for normal entry.
- LIVE seat selection now works with a real pointer. The pressed animation preserves the table seat's positioning transform instead of moving the button away before the click completes.
- Empty seats show explicit seat numbers, larger interactive targets, loading feedback, accessible labels, and a confirmation after the player sits down.
- Lobby seats are no longer incorrectly marked as “acting” before a hand starts, and ready controls clearly wait for a second player instead of offering an action that cannot start a hand.
- The lobby received responsive two-column and single-column layouts, live table summaries, participant avatars, stable hover states, and 375px mobile tuning.

## Verification

- Formatting, TypeScript checks, production builds, 30 poker-engine tests, 32 server tests, and 17 web tests pass on Node.js 24.
- Real PostgreSQL, HTTP, and Socket.IO tests cover the existing three-player ONLINE flow, shared lobby discovery, two-player direct LIVE join, LIVE seat selection, all four betting streets, result objection/replacement/confirmation, and concurrent six-seat capacity.
- Browser verification covers account initialization, shared lobby rendering, direct LIVE entry, real pointer seat selection, ready-state copy, desktop layout, and 375px mobile overflow/touch targets.

## Database and rollback

This release does not add a database migration. Rolling back to v0.3.1 does not require a database rollback, but v0.3.1 clients will return to invitation-oriented lobby behavior.
