# Poker with Friends 文案规范

## 语气

- 清晰：优先说明当前状态或可执行动作。
- 克制：不使用宣传口号、拟人表达或重复说明。
- 友好：使用自然中文，不暴露内部技术术语。

## 统一术语

| 使用       | 不使用               |
| ---------- | -------------------- |
| 牌桌       | 房间、朋友桌、这桌   |
| 管理员     | 开桌人、组局小助手   |
| 牌谱       | 记录、回看、牌局回放 |
| 筹码       | 积分                 |
| 下一手暂离 | 下手休息             |
| 准备下一手 | 人齐再开、我准备好了 |

## 规则

- 按钮使用动词或“动词＋对象”，不使用句子。
- 状态提示只描述当前状态；需要用户处理时再补充下一步。
- 错误提示不出现令牌、快照、序列等内部实现词汇。
- 首页之外不使用宣传性文案；品牌名称中的 `Friends` 不受此限制。
- 输入规则、下注金额含义和不可逆操作影响必须保留。

## 示例

| 场景     | 文案                       |
| -------- | -------------------------- |
| 等待开局 | 等待玩家准备               |
| 已确认   | 已准备，等待其他玩家       |
| 阵容变化 | 牌桌成员已变化，请重新准备 |
| 状态过期 | 牌桌状态已更新，请重试     |
| 加载失败 | 牌谱加载失败               |
| 争议     | 结果存在异议，本手已暂停   |
